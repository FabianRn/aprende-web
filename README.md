# 🚀 Aprende Web — Mi camino a desarrollador Fullstack

Repositorio donde guardo los proyectos de mi aprendizaje de desarrollo web, desde los fundamentos hasta proyectos completos.

## 📂 Proyectos

### 01 — Mi primera página (`01-primera-pagina/`)
Primera página web. Práctica de **HTML** (estructura, listas, enlaces), **CSS** (estilos, colores, diseño) y **JavaScript** (eventos, interactividad).

### 02 — Lista de Tareas (`02-lista-tareas/`)
Aplicación de tareas (to-do list) completa y funcional:
- ➕ Agregar y borrar tareas
- ✅ Marcar tareas como completadas (tachado)
- 💾 Guardado permanente con `localStorage` (las tareas no se pierden al recargar)
- 🔢 Contador de tareas pendientes

**Tecnologías:** HTML5, CSS3, JavaScript (DOM, arrays, objetos, localStorage, JSON).
